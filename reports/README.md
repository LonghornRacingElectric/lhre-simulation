# Reports

Reports are separated at the level where the design story naturally branches:

- `2026-vdyn-design-report.md`
- `2026-aero-design-report.md`
- `2026-chassis-design-report.md`
- `2026-judge-question-bank.md`
- `2026-design-report-index.md`
- `2026-design-event-comprehensive-analysis.md`

Each report should include only results from purpose-defined studies that have
been run and committed as `RESULTS.md` artifacts. If a claim does not have a
study behind it yet, it belongs in the open-questions section, not the findings
section.
