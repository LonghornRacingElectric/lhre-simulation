# DS-006 Integrated Tire Design

This study compares every Round 9 full-UM14 tire fit on the current vehicle
architecture using both EnvelopeSim and StandardSim SteadyStateEval.

Run from the repository root:

```bash
/tmp/lhre-sim-venv/bin/python studies/DS-006-integrated-tire-design/run.py --reuse
```

For a quick EnvelopeSim-only report:

```bash
/tmp/lhre-sim-venv/bin/python studies/DS-006-integrated-tire-design/run.py --skip-standardsim
```
