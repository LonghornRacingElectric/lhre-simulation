# Round 9 Tire Fit Assumptions And PAC2002 Formulation

Generated by `tire_fits/fit_round9_tires.py`.

## Scope

This fitting pass creates BobSim-compatible MF5.2/PAC2002-style `.tir` files
from the Round 9 TTC Matlab SI archives and `RunGuide_Round9.pdf`.

- Generated files: `55`
- Direct drive/brake/combined fits: `39`
- 16 inch Hoosier files with scaled 18 inch donor longitudinal/combined terms: `16`
- Use mode: `USE_MODE = 4`, steady combined force/moment behavior

## Data Interpretation

- Normal load uses `Fz = -FZ` because the raw TTC channel is negative in
  compression.
- Slip angle `SA` and inclination angle `IA` are interpreted as degrees and
  converted to radians before fitting.
- Slip ratio uses `SR` from the drive/brake archive.
- Pressure `P` is interpreted as kPa. A run contributes to a pressure-specific
  fit when `abs(P - psi * 6.894757) <= 2.35 kPa`.
- Speed `V` is interpreted as km/h for filtering. Fits use samples with
  `V > 18 km/h`.
- Fitted files use `LONGVL = 40 km/h = 11.111 m/s`.
- Each pressure is fit as a separate `.tir`; no smooth pressure-polynomial
  model is identified.

## Run Selection

The script encodes the Round 9 RunGuide matrix. Warmup and transient-only runs
are excluded. The Goodyear D2704 20.0x7.0-13 on 8 inch rim has no usable 8 psi
cornering block in the available RunGuide/data combination, so that one file is
not generated.

## Pure Lateral PAC2002 Form

The fitted pure lateral coefficients are:

`PCY1, PDY1, PDY2, PDY3, PEY1, PEY2, PEY3, PEY4, PKY1, PKY2, PKY3, PHY1, PHY2, PHY3, PVY1, PVY2, PVY3, PVY4`

The formulation matches the BobLib MF5.2/PAC2002 implementation:

```text
dfz = (Fz - FNOMIN) / FNOMIN
gamma_y = gamma
mu_y = (PDY1 + PDY2*dfz) * (1 - PDY3*gamma_y^2)
C_y = PCY1
D_y = mu_y * Fz
K_y = PKY1*FNOMIN*sin(2*atan(Fz/(PKY2*FNOMIN))) * (1 - PKY3*abs(gamma_y))
B_y = K_y / (C_y*D_y)
S_Hy = PHY1 + PHY2*dfz + PHY3*gamma_y
S_Vy = Fz*((PVY1 + PVY2*dfz) + (PVY3 + PVY4*dfz)*gamma_y)
alpha_y = alpha + S_Hy
E_y = (PEY1 + PEY2*dfz) * (1 - (PEY3 + PEY4*gamma_y)*sign(alpha_y))
Fy0 = D_y*sin(C_y*atan(B_y*alpha_y - E_y*(B_y*alpha_y - atan(B_y*alpha_y)))) + S_Vy
```

## Pure Longitudinal PAC2002 Form

The fitted pure longitudinal coefficients are:

`PCX1, PDX1, PDX2, PDX3, PEX1, PEX2, PEX3, PEX4, PKX1, PKX2, PKX3, PHX1, PHX2, PVX1, PVX2`

```text
dfz = (Fz - FNOMIN) / FNOMIN
gamma_x = gamma
mu_x = (PDX1 + PDX2*dfz) * (1 - PDX3*gamma_x^2)
C_x = PCX1
D_x = mu_x * Fz
K_x = Fz*(PKX1 + PKX2*dfz)*exp(PKX3*dfz)
B_x = K_x / (C_x*D_x)
S_Hx = PHX1 + PHX2*dfz
S_Vx = Fz*(PVX1 + PVX2*dfz)
kappa_x = kappa + S_Hx
E_x = (PEX1 + PEX2*dfz + PEX3*dfz^2) * (1 - PEX4*sign(kappa_x))
Fx0 = D_x*sin(C_x*atan(B_x*kappa_x - E_x*(B_x*kappa_x - atan(B_x*kappa_x)))) + S_Vx
```

## Combined Slip Form

Combined longitudinal force uses the PAC2002 reduction form:

```text
B_xa = RBX1*cos(atan(RBX2*kappa))
C_xa = RCX1
E_xa = REX1 + REX2*dfz
alpha_s = alpha + RHX1
G_xa = cos(C_xa*atan(B_xa*alpha_s - E_xa*(B_xa*alpha_s - atan(B_xa*alpha_s))))
      / cos(C_xa*atan(B_xa*RHX1 - E_xa*(B_xa*RHX1 - atan(B_xa*RHX1))))
Fx = Fx0 * G_xa
```

Combined lateral force uses the reduction plus kappa-induced side-force shift:

```text
B_yk = RBY1*cos(atan(RBY2*(alpha - RBY3)))
C_yk = RCY1
E_yk = REY1 + REY2*dfz
S_Hyk = RHY1 + RHY2*dfz
kappa_s = kappa + S_Hyk
G_yk = cos(C_yk*atan(B_yk*kappa_s - E_yk*(B_yk*kappa_s - atan(B_yk*kappa_s))))
      / cos(C_yk*atan(B_yk*S_Hyk - E_yk*(B_yk*S_Hyk - atan(B_yk*S_Hyk))))
D_Vyk = mu_y*Fz*(RVY1 + RVY2*dfz + RVY3*gamma)*cos(atan(RVY4*alpha))
S_Vyk = D_Vyk*sin(RVY5*atan(RVY6*kappa))
Fy = Fy0 * G_yk + S_Vyk
```

## 16 Inch Hoosier Longitudinal Scaling

The Round 9 RunGuide does not provide 16 inch Hoosier drive/brake/combined
runs. For those tires:

- Pure lateral coefficients are still direct 16 inch fits.
- The donor is the nearest same-pressure Hoosier 43100 18.0x6.0-10 R20 fit.
- Donor combined-slip shape terms are preserved.
- Donor longitudinal peak terms are scaled to preserve the donor long/lateral
  friction ratio:

```text
donor_ratio = abs(PDX1_donor) / abs(PDY1_donor)
target_PDX1 = abs(PDY1_target) * donor_ratio
mu_scale = target_PDX1 / abs(PDX1_donor)
PDX1, PDX2, PVX1, PVX2 <- donor value * mu_scale
```

- Longitudinal stiffness terms are scaled by the target/donor lateral stiffness
  ratio:

```text
K_y_nom = abs(PKY1*FNOMIN*sin(2*atan(1/PKY2)))
stiffness_scale = K_y_nom_target / K_y_nom_donor
PKX1, PKX2 <- donor value * stiffness_scale
```

The manifest records `longitudinal_mu_scale` and
`longitudinal_stiffness_scale` for every generated file.

## Optimization

- Optimizer: `scipy.optimize.least_squares`.
- Loss: `soft_l1`, to reduce sensitivity to bad sweeps, warmup artifacts, and
  transient points that survive filtering.
- Residuals are normalized by a load-scaled force floor.
- A deterministic random downsample caps each fit channel at 5000 samples.

## What Is Not Claimed

- This is not a full commercial TTC/PAC2002 fit with every moment, pressure,
  thermal, speed, and relaxation dependency identified simultaneously.
- Aligning torque, rolling resistance, and relaxation terms are not newly fit
  from Round 9 in this script; unsupported terms are inherited from the current
  reference tire or kept simple.
- The 16 inch Hoosier longitudinal/combined behavior is inferred from 18 inch
  Hoosier data, not directly measured.
