# OpenLAP attribution

`src/openlap_solver.py` is a headless Python execution port of equations from
Michael Halkiopoulos's OpenLAP Lap Time Simulator, cloned under
`vendor/OpenLAP-Lap-Time-Simulator`.

OpenLAP is licensed under GNU GPL v3. The execution port is distributed under
GPL-3.0-or-later. The untouched upstream license is at
`vendor/OpenLAP-Lap-Time-Simulator/LICENSE`.
